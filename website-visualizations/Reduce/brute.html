🔴 Brute Force Reduce Simulator

Algorithm Description:

Baseline (Brute-Force) Reduce: The root process (0, 0) sequentially receives messages from all N² − 1 other processes, one at a time.

Routing: From each source (i, j), the data travels hop-by-hop to (0,0): (i,j) → ... → (i,0) → ... → (0,0)

Operation: Each node sends its local value (1) to the root, which accumulates the sum.

  • 🟢 Green: Root node (0,0) - accumulates the sum
  • 🟠 Orange: Current source node sending data
  • 🔵 Blue: Intermediate routing hops
  • ⚪ Gray: Completed (data already sent)
