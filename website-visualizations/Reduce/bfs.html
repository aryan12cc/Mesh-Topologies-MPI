🧮 Inclusion-Exclusion Reduce Simulator

Algorithm Description:

Inclusion-Exclusion Reduce: Each node waits for data from its West, North, and North-West diagonal neighbors, then computes:

result = local_value + value_from_west + value_from_north - value_from_north_west_diagonal

This formula follows the inclusion-exclusion principle to avoid double-counting.

Wave Pattern: The computation propagates in diagonal waves from bottom-right (N-1, N-1) to top-left (0, 0). All nodes in the same diagonal can compute in parallel once their dependencies arrive.

3D Extension: Similar principle starting from (N-1, N-1, N-1) and propagating to (0, 0, 0) with 7 incoming dependencies.
