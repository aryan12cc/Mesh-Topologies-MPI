📊 Row-Column Pipeline Reduce Simulator

Algorithm Description:

Row-Column Pipeline Reduce: The reduction is performed in two phases.

Phase 1 (Vertical/Column Reduction): Within each column, data flows from bottom to top (south to north). Each topmost node (row 0) accumulates its column's sum and becomes a column leader.

Phase 2 (Horizontal/Row Reduction): The column leaders (row 0) send their accumulated values leftward (east to west) to the root (0,0).

3D: First reduce along Z (back→front), then Y (south→north), then X (east→west) dimensions.
