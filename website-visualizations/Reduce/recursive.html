🔄 Recursive / Quadrant-Based Reduce Simulator

Algorithm Description:

Recursive / Quadrant-Based Reduce: The grid is recursively split into quadrants (2D) or octants (3D). Each quadrant performs reduction internally, then sends its result to the parent region's root.

2D Example (4×4): Each quadrant reduces internally first. Then quadrant roots at (2,0), (0,2), and (2,2) send their results to (0,0). This happens recursively at each level.

3D: Each octant reduces internally, then all 7 non-root octants send to the root octant: from (+x,0,0), (0,+y,0), (+x,+y,0), (0,0,+z), (+x,0,+z), (0,+y,+z), (+x,+y,+z) → (0,0,0).

  • 🟧 Orange Border: Quadrant/Octant roots

  • ⚪ Gray: Completed
