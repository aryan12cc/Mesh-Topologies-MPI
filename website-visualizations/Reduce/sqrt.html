🔲 Hierarchical / Block-Based Reduce Simulator

Algorithm Description:

Hierarchical / Block-Based Reduce: The N × N (or N × N × N) grid is divided into k blocks (e.g., √N blocks of size √N × √N).

Phase 1 (Intra-block): Each block performs local reduction using pipeline.

2D: Within each block: south→north, then east→west. Each block's leader (top-left corner) accumulates its block's sum.

3D: Within each block: back→front (Z), then south→north (Y), then east→west (X). Block leader at corner (0,0,0) of each block accumulates.

Phase 2 (Inter-block): All block leaders send their accumulated sums to the global root (0,0,0).
