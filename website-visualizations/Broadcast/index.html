🚀 MPI Broadcast Algorithms

Interactive Visualizer for 2D & 3D Mesh Topologies

About This Project

This interactive visualizer demonstrates various MPI collective communication algorithms for broadcast operations on 2D and 3D mesh topologies. Each algorithm implements a different strategy for efficiently distributing data from a root process to all other processes in the network.

📊 Key Concepts: These algorithms differ in terms of message complexity, latency, and implementation strategy. Some prioritize minimizing the number of hops, while others focus on exploiting parallelism through pipelining or hierarchical structures.

Choose an Algorithm to Visualize

🔴

Brute Force

Sequential routing from root (0,0) to each node individually using dimension-order routing.

  • Simple baseline implementation
  • High message complexity: O(N³)
  • Sequential execution
  • 2D & 3D support
 📡

Row-Column Pipeline

Dimension-order pipeline broadcast. 2D: row then column. 3D: X, Y, then Z axis.

  • Pipelined execution
  • Message complexity: O(N)
  • Two phases (2D) / Three phases (3D)
  • Exploits parallelism
 🌊

BFS / Spanning Tree

Breadth-first search approach where each node forwards to unvisited neighbors.

  • Natural spanning tree
  • Low latency: O(2N)
  • Message redundancy handled
  • Fast wavefront propagation
 🔄

Recursive Quadrant

Recursively divides grid into quadrants/octants. Root sends to sub-roots which recurse.

  • Divide-and-conquer strategy
  • Logarithmic depth: O(log N)
  • Balanced tree structure
  • Efficient for large grids
 🔲

Hierarchical (Block-Based)

Divides grid into √N blocks. Phase 1: broadcast to block leaders. Phase 2: intra-block broadcast.

  • Two-phase approach
  • Block-level organization
  • Combines inter & intra-block
  • Scalable design
