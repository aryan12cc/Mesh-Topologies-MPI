🔲 Hierarchical / Block-Based Broadcast Simulator

Algorithm Description:

Hierarchical / Block-Based Broadcast: The N × N grid is conceptually divided into k blocks (e.g., √N blocks of size √N × √N).

Phase 1: The global root (0, 0) broadcasts the message to the "leader" process of all k blocks.

Phase 2: Each block leader broadcasts the message to all processes within its own block, using Row-Column pipeline method.

  • 🟣 Purple: Phase 2 (Intra-block broadcast)
  • 🟢 Green: Completed
