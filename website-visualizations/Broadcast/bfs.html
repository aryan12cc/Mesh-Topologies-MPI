🌊 BFS / Spanning-Tree Broadcast Simulator

Algorithm Description:

BFS / Spanning-Tree Broadcast: The root (0, 0) sends to its neighbors, e.g., (0, 1) and (1, 0). Each process, upon receiving the message for the first time, forwards it to its neighbors (e.g., right and down) that have not received it. Consequent messages are ignored.

  • 🔴 Red: Root node (0,0,0)
  • 🔵 Blue: Active (just received, will forward)
  • 🟢 Green: Completed (message forwarded)

Note: Each node forwards to neighbors (right, down, and forward in 3D) only once upon first reception.
