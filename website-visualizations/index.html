📡 Distributed Systems Visualizations

Interactive visualizations of collective communication algorithms on mesh networks

About This Project

This project provides interactive visualizations of fundamental collective communication operations in distributed systems, specifically designed for 2D and 3D mesh network topologies.

Collective operations are essential building blocks in parallel and distributed computing, used in applications ranging from scientific simulations to machine learning. Understanding how data flows through a mesh network helps in optimizing performance and minimizing communication overhead.

Each visualization demonstrates different algorithmic approaches with varying time complexities, allowing you to compare their behavior step-by-step.

📤

Broadcast (Bcast)

One process (the root) sends the same message to all other processes in the network. The message propagates from a single source to all N² (2D) or N³ (3D) nodes.

Use Cases: Configuration distribution, model synchronization, parameter updates
Brute Force BFS-Based Row-Column Pipeline Recursive Quadrant Hierarchical Block
📥

Reduce

All processes send their local data to a single root process, which combines (reduces) all values into a single result. This is the reverse operation of broadcast.

Use Cases: Sum aggregation, gradient accumulation, global statistics computation
Brute Force Inclusion-Exclusion Row-Column Pipeline Recursive Quadrant Hierarchical Block

Features

  • Interactive Controls: Step through algorithms one round at a time or play continuously
  • 2D & 3D Support: Switch between 2D mesh grids and 3D mesh cubes
  • Configurable Grid Size: Adjust N to see how algorithms scale
  • Visual Feedback: Color-coded nodes show message flow, routing paths, and completion states
  • Performance Metrics: Track round count and algorithm progress in real-time
  • Multiple Algorithms: Compare 5 different approaches for each operation
